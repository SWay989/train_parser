# Парсер расписания электричек

Этот проект позволяет извлечь расписание электричек со страницы [tutu.ru](https://www.tutu.ru/station.php?nnst=45807) и сохранить его в JSON.

## Что делает программа
- Парсит HTML-страницу станции.
- Извлекает для каждого рейса:
  - Время отправления (HH:MM)
  - Маршрут в формате `Откуда → Куда`
  - Дни рейса (`будни`, `ежедневно`)
- Позволяет фильтровать рейсы по типу дней.
